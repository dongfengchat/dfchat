02 · 核心功能 为什么是东风快信

把团队沟通的一切都装进一个客户端

不用切换应用、不用复制粘贴文件、不用为了视频会议另外下软件。

实时聊天

私聊、群组、Discord 风格的多频道服务器，断线自动重连，离线消息自动补齐。

高清通话

一对一语音 + 视频 + 屏幕共享，WebRTC 直连，画面 720p 起步。

文件传输

拖拽上传、粘贴上传、最高 100MB 单文件。图片自动缩略图。

全文搜索

⌘K 一键搜遍所有会话历史，中英文都能搜。点击直接跳到原消息。

桌面通知

窗口未聚焦时系统通知 + Dock 角标，免打扰模式按会话级别设置。

表情反应

给消息加 emoji 反应、引用回复、撤回、@提及 + 高亮、消息置顶。

账号安全

JWT + 刷新令牌、多设备会话管理、强制下线、改密码、黑名单、注销账号。

群组权限

群主 / 管理员 / 成员三级 RBAC，可建频道、删频道、踢人、升降管理员。

端到端 TLS

所有流量走 HTTPS / WSS，文件存储 MinIO 强加密，密码 bcrypt 12 轮。

macOS

这是系统的 Gatekeeper 在阻止未签名应用（我们的 Apple Developer 证书正在准备中）。请按以下步骤跳过：

  1. 把 东风快信.app 拖入「应用程序」文件夹
  2. 打开「终端」（Spotlight 搜索 Terminal），粘贴这条命令并回车： 
    sudo xattr -dr com.apple.quarantine /Applications/东风快信.app
  3. 输入开机密码（屏幕不会显示，盲打）回车
  4. 正常双击启动东风快信

仍报「已损坏」或闪退？再执行一次：

sudo codesign --force --deep --sign - /Applications/东风快信.app

Windows

双击 DFCHAT-win-x64.exe，若弹出 SmartScreen 蓝框：

  1. 点击蓝框里的 "更多信息"
  2. 点击下方出现的 "仍要运行" 按钮

之后是常规安装向导，下一步到底即可。装完桌面会出现快捷方式，开始菜单也有「东风快信」。

Defender 误删 exe？打开「Windows 安全中心 → 病毒和威胁防护 → 保护历史记录」恢复并加白名单。

正式版会附带 Apple Developer ID 公证 + Windows OV 代码签名，届时所有警告会自动消失。

05 · 使用指南 使用指南

详细功能说明

点击每张卡片展开。读完一遍大约 8 分钟，能熟练用上 90% 的功能。

注册与登录

客户端首屏点击 「立即注册」，或在本网站下方 免费注册 区域填表。

  • 用户名：3–32 位字母 / 数字 / 下划线，用于登录与好友查找，注册后不能改
  • 邮箱：用于未来找回密码（暂不支持自助找回）
  • 密码：至少 8 位，密码框旁的 👁 可切换显隐
  • 昵称：可选，显示给好友看，可随时修改

注册成功后客户端自动登录，下次启动保持登录状态。多设备同时登录互不冲突，每台设备一个独立 token。

忘记密码或被踢出？联系群主或管理员重置（在「设置 → 安全」也能强制下线其他设备）。
添加好友 / 加入群组

侧栏顶部 「+」 按钮聚合所有入口。

  • 添加好友：输入对方用户名 → 发送好友请求，对方接受后出现在侧栏好友列表
  • 创建群组：填群名称、公开/私密、最大人数 → 系统自动生成邀请码分享给朋友
  • 加入群组：拿到邀请码后粘贴即可
  • 群内频道：Discord 风格，群主 / 管理员可建多个频道（公开 / 私密 / 通知型）

角色三级：群主（唯一，可解散）/ 管理员（可踢人、删消息、建/删频道）/ 成员。群主可右键群成员升降管理员。

发消息

基础：输入框打字，↵ 发送，Shift + ↵ 换行。

  • @ 提及：输入 @ 弹出好友 / 成员列表，被提及者高亮 + 桌面通知（即使免打扰也会响）
  • 引用回复：右键消息 → 引用，对方点击引用框直接跳到原消息
  • 编辑：自己的消息右键 → 编辑（5 分钟内）
  • 撤回：自己的消息右键 → 撤回（无时限，对方看到「消息已撤回」占位）
  • 表情反应：消息悬停时右下角出现笑脸图标 → 选 emoji，多人可叠加同一表情
  • 置顶：群主 / 管理员右键消息 → 置顶，每个会话最多 5 条
支持 Markdown 简写：**粗体** · *斜体* · `代码` · 三个 backtick 包代码块。
文件 / 图片传输

三种上传方式：

  1. 点击输入框旁的 回形针图标 → 选择文件
  2. 拖拽文件直接进聊天窗口
  3. 截图 / 复制图片后 ⌘V / Ctrl V 粘贴

单文件最大 100 MB。图片自动生成缩略图，点击查看大图 + 滚轮缩放。文件存储走 MinIO 对象存储，预签名直传，不经过 API 服务器，速度受用户带宽决定。

直传 7 天后老链接会失效，但消息记录里的图片永久可访问（每次重新签发）。
语音 / 视频通话

在好友会话右上角点 📞 语音 / 📹 视频按钮发起。

  • 对方收到来电弹窗（即使最小化也会响铃 + 任务栏闪动）
  • 接通后可：静音、关摄像头、屏幕共享、挂断
  • 屏幕共享：通话中点共享按钮，选「整个屏幕」或「单个窗口」，无需重新协商连接

底层走 WebRTC P2P 直连，画质 720p+。对称 NAT 用户自动 fallback 到 TURN 中继（coturn 服务器），公司内网 / 4G 等场景也能打通。

目前只支持 1 对 1 通话。多人会议（SFU）在路线图里。
直播 + 弹幕

侧栏 📡 直播图标进入直播模块。

  • 广场：浏览所有正在直播的房间，点封面进观看
  • 我的直播：开播一个新房间，复制：
    · 服务器 URL：rtmp://dfchat.chat/live
    · 推流密钥：系统生成（可一键旋转）
    填进 OBS 的「直播 → 自定义」即可
  • 观看：HLS 自适应，弹幕飘过画面 + 右侧时间轴列表
  • 历史回放：直播自动 DVR 录制，结束后在该房间下方点开回看
推流码千万别泄露，等于把你房间的直播权交给别人。怀疑泄露立即「旋转密钥」生成新的。
全局搜索

任何页面按 ⌘K（Mac）/ Ctrl K（Win / Linux）打开搜索面板。

  • 消息内容（中英文都支持，PostgreSQL 全文索引）
  • 联系人 / 群组名 / 频道名
  • 文件名（按附件名搜）

点击结果直接跳到该消息上下文，前后各 10 条自动加载。搜索范围 = 所有你能看到的会话历史（不会跨越群组权限）。

设置 / 账号安全

左下角头像 → 设置。

  • 个人资料：头像、昵称、个人简介
  • 安全：修改密码、查看登录设备 / 强制下线、注销账号
  • 通知：免打扰按会话级别设置，群组可设「仅 @ 提及」
  • 外观：暗色 / 自动主题

多设备会话：同时在多台电脑登录互不冲突。「安全」页可看到所有活跃设备 + 一键下线异常设备。

注销账号会永久删除所有数据：消息、文件、好友关系，无法恢复。
快捷键速查
  • ⌘K / Ctrl K — 全局搜索
  • ⌘, / Ctrl , — 打开设置
  • ↵ — 发送消息
  • Shift ↵ — 输入框换行
  • ↑（输入框空时）— 编辑上一条自己的消息

  • : — 触发 emoji 补全（如 :smile:）
  • ⌘V / Ctrl V 在聊天框 — 粘贴图片直接发送

  • M — 静音切换
  • V — 关 / 开摄像头
  • ⌘Shift H / Ctrl Shift H — 挂断
