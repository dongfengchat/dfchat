# Contributor Covenant Code of Conduct

## 我们的承诺

为了营造一个开放友好的环境，我们作为贡献者和维护者承诺：让参与本项目和我们社区的每个人都不会受到骚扰，无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育水平、社会经济地位、国籍、外貌、种族、宗教或性别取向。

## 我们的标准

促进社区积极环境的行为举例：

- 对他人展现共情和善意
- 尊重不同意见、观点和经验
- 给予并大方接受建设性反馈
- 承担责任并向受我们错误影响的人道歉，且从经验中学习
- 关注的不仅是个体最优，而是整个社区的最优

不可接受的行为举例：

- 使用性化的言语或图像，以及任何形式的性关注或挑逗
- 挑衅、侮辱性或贬损性评论以及个人或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息，如住址或电子邮件
- 在专业场合中合理认定为不当的其他行为

## 执行责任

社区领导者负责澄清并执行可接受行为的标准，并对任何他们认为不当、威胁、冒犯或有害的行为采取适当且公平的纠正措施。

## 范围

本行为准则适用于所有社区场所，也适用于个人在公共空间正式代表社区的情况。

## 执行

辱骂、骚扰或其他不可接受的行为可通过 **dongfengchat@gmail.com** 报告给社区领导者。所有投诉将得到迅速且公平的审查和调查。所有社区领导者有义务尊重事件报告者的隐私和安全。
